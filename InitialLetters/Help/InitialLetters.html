Letter Positions and Distances

Initial letters shift their chapter’s text by a number of letters, words, and even verses as shown in chapter 42. The order of initial letters is also part of the code so (حم) is not the same as (مح) that relies on distances between the same letters. This means that both letter positions and same-letter distances are part of the Quran code. Letter position can be within chapter, verse and/or word, while same-letter distances is measured in how many letters, words, and/or verses are in between same letters.
